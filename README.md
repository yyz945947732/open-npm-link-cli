# open-npm-link-cli

> Open a npm link for target package

## Install

```bash
npm install --global open-npm-link-cli
```

## Usage

```txt
$ onpm --help

  Usage
    $ onpm [options]

  Options
    -h --help              Print this help
    -v --version           Print open-npm-link-cli version number
    -b --browser           Browser to open

  Examples
    $ onpm lodash
    $ onpm lodash -b "chrome"
```
